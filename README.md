# 🍰 Sonam's Bakery Website

# 

# "Mom's Touch, Healthy bhi, Tasty bhi 😋"

# 

# 📌 About the Project

# 

# This website is created for a local home bakery called Sonam’s Bakery, based in Ahmedabad.

# The main idea was to give the bakery a simple online presence where customers can check products and easily place orders.

# 

# This project was done as part of the Future Interns Web Development Internship.

# 

# \---

# 

# 🚀 Features

# 

# \- Responsive design (works on mobile and desktop)

# \- Simple navigation with sticky navbar

# \- Menu section with category filter (cakes, cupcakes, cookies, healthy items)

# \- WhatsApp order option for quick ordering

# \- Contact form for customer details

# \- Image gallery and customer reviews

# \- Basic animations for better UI

# 

# \---

# 

# 🛠️ Technologies Used

# 

# \- HTML

# \- CSS

# \- JavaScript

# 

# \---

# 

# 🎯 Purpose

# 

# The goal of this project is to help a small bakery:

# 

# \- Reach more customers online

# \- Show their products clearly

# \- Make ordering easier using WhatsApp

# 

# \---

# 

# 📱 WhatsApp Integration

# 

# Customers can directly place orders through WhatsApp.

# Clicking the order button opens WhatsApp with a ready message, making it simple for both customer and business.

# 

# \---

# 

# 📂 Project Structure

# 

# \- "index.html" → main webpage

# \- "styles.css" → styling and layout

# \- "script.js" → functionality and interactions

# 

# \---

# 

# 🚀 How to Run

# 

# 1\. Download all files

# 2\. Open "index.html" in any browser
